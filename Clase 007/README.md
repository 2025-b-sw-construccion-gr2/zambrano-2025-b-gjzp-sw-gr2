# Proyecto: Mapa Mental con Markmap

## Objetivo

Se generaró un mapa mental interactivo a partir de Markdown utilizando Markmap.

## Pasos

- Instalar la extensión o utilizar la CLI.
- Escribir la estructura del mapa mediante encabezados.
- Abrir la vista previa o exportar a HTML/PDF según se prefiera.

## Recursos

- Extensión Markmap para VS Code
- [Markmap](https://markmap.js.org/)

## Notas

- Las listas y el texto bajo cada encabezado aparecen como nodos.
- Se pueden incrustar fragmentos de código y enlaces; Markmap los muestra en nodos expandibles.
- Se han añadido versiones en HTML y PDF para que quien desee pueda leer en el formato que prefiera.
